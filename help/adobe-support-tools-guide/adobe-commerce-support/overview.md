# サポートツールの概要

Adobe Commerceでは、e コマースストア体験を向上させるのに役立つ、さまざまなサポートツールを提供しています。 パーソナライズされたベストプラクティス、診断および監視ツール、サイトに関する最適な情報を提供します。

* サイト全体の分析ツール

   * [Adobeによるサイト全体の分析ツール：概要](https://experienceleague.adobe.com/ja/docs/commerce-operations/tools/site-wide-analysis-tool/intro)

* [!DNL Quality Patches Tool] （QPT）

   * [QPT ツールの概要で使用可能なパッチ](https://experienceleague.adobe.com/ja/docs/commerce-operations/tools/quality-patches-tool/patches-available-in-qpt/patches-available-in-qpt-tool-overview)

     [[!DNL Quality Patches Tool]](https://github.com/magento/quality-patches) （QPT）は、AdobeおよびMagento Open Source コミュニティによって開発された個別の品質向上パッチを適用できるツールです。

     これにより、次のことが可能になります。

      * パッケージに含まれている品質パッチを適用する
      * 以前に適用したパッチを復元する
      * インストールされているバージョンのAdobe Commerceで使用できる高品質パッチに関する一般的な情報を表示します。

     このツールは、Adobe Commerceで発生する可能性のある問題のパッチをセルフサービスで使用したり、Adobe Commerce サポートによって提案されたパッチを簡単に適用したりできるようにすることを目的としています。
