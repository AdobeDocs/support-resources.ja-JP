# Admin Consoleでパスワードの有効期限を強制する

> **適用先：** エンタープライズおよびチーム

組織システム管理者は、このプロセスに従って、ユーザーのパスワードの有効期限を設定できます。

## 環境

[Adobe Admin Console](https://adminconsole.adobe.com/)

## 手順

1. 「[Adobe Admin Console](https://adminconsole.adobe.com/settings)**[!UICONTROL 設定]**」セクションに移動し、左上の「**[!UICONTROL プライバシーとセキュリティ]**」オプションを展開します。

1. **[!UICONTROL 認証設定]** を選択します。

1. **[!UICONTROL 認証レベル]** を確認し、ユーザーに最も適した認証レベル（**[!UICONTROL ユーザーにとって最も簡単]**、**[!UICONTROL より安全]**、**[!UICONTROL 最も安全]** を選択します。

使用可能な認証およびセキュリティポリシーについて詳しくは、[&#x200B; 認証設定 Helpx](https://helpx.adobe.com/enterprise/using/authentication-settings.html) を参照してください。
