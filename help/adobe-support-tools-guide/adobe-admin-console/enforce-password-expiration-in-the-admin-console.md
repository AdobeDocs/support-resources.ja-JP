# Admin Consoleでのパスワードの有効期限の適用

**適用先：** エンタープライズ版とグループ版

組織システム管理者は、このプロセスに従って、ユーザーのパスワード有効期限を設定できます。 [Adobe Admin Console](https://adminconsole.adobe.com/)にログインします。

## 手順

1. [Adobe Admin Console](https://adminconsole.adobe.com/settings) **[!UICONTROL Settings]** セクションに移動し、左上の&#x200B;**[!UICONTROL プライバシーとセキュリティ]** オプションを展開します。

1. **[!UICONTROL 認証設定]**&#x200B;を選択します。

1. **[!UICONTROL 認証レベル]**&#x200B;を確認し、ユーザーに最適な認証レベルを選択します：**[!UICONTROL ユーザーにとって最も簡単]**、**[!UICONTROL より安全]**、**[!UICONTROL 最も安全]**。

使用可能な認証ポリシーとセキュリティポリシーについて詳しくは、[認証設定Helpx](https://helpx.adobe.com/enterprise/using/authentication-settings.html)にアクセスしてください。
