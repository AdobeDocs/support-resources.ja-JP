# 保留中のジョブの実行

この機能は、[[!DNL Global Admin Console]](https://global-admin-console.adobe.com/)を使用しているエンタープライズ組織に適用されます。

- [[!DNL Global Admin Console]](https://global-admin-console.adobe.com/)の変更は、次の2つのフェーズで完了します。

   1. **フェーズを編集**：組織に変更を加えるか、製品を割り当てます。
   2. **実行フェーズ**：保留中の変更を確認して実行し、変更を有効にします。

- [[!DNL Global Admin Console]](https://helpx.adobe.com/enterprise/global-admin-console/adopt-global-administration.html)で行われたすべての変更が実装され、有効になっていることを確認するには、「**[!UICONTROL ジョブ実行]**」タブを選択し、保留中の変更の実行に進みます。

  [[!DNL Global Admin Console]](https://global-admin-console.adobe.com/)にログインします。

## 変更の永続性と可視性

### 永続性の変更

- 保留中の変更を失うことなく、後でログアウトして返品できます。
- 未実行の変更：
   - 30日後に破棄されます。
   - ブラウザーのタブやウィンドウを閉じるなど、セッションが終了するとクリアされます。

> [!NOTE]
>
> 重要な変更を迅速に実行し、適切に適用できます。

### 複数の管理者と競合

- 同じ組織で作業している2人の管理者：
   - お互いの未実行の変更が表示されない。
   - 次の後でのみ変更を表示：
      - 実行、および
      - ディスプレイを更新するか、再度ログインします。
- 未実行の変更は、既に実行された変更と競合する可能性があります。

### 対立の処理

競合が報告されます：
- 実行時、または
- データが更新されたとき（ログアウトして再度ログインした後など）。

## 保留中の変更の実行

グローバル管理者は、「**[!UICONTROL ジョブ実行]**」タブから保留中の変更を確認して実行できます。

### 変更を実行する手順

1. [!DNL Global Admin Console]にログインします。
2. 左側のナビゲーションパネルから「**[!UICONTROL ジョブ実行]**」タブを選択します。
3. 保留中の変更を確認します。
4. **[!UICONTROL 変更を送信]**&#x200B;を選択して実行します。

ジョブを送信した後：

- ジョブが実行キューに入ります。
- ジョブの実行中にステータスは&#x200B;**[!UICONTROL 保留中]**&#x200B;です。
- Adobeでは、予測可能性とトラブルシューティングを容易にするために、一度に1つのジョブのみを実行することをお勧めします。

> [!IMPORTANT]
>
> 実行中にエラーが発生した場合、正常に適用されなかった変更は再入力して再送信する必要があります。

### 長時間実行中の割り当て

製品配分に12時間以上かかる場合：
- ジョブは&#x200B;**[!UICONTROL 失敗]**&#x200B;としてマークされています。
- そのジョブ内のその後の保留中のタスクは実行されません。

## 実行中のジョブのキャンセル

現在実行中のジョブは、**[!UICONTROL ジョブ実行]** タブからキャンセルできます。

### 実行中のジョブをキャンセルする手順

1. [!DNL Global Admin Console]にログインします。
2. 「**[!UICONTROL ジョブ実行]**」を選択します。
3. 「**[!UICONTROL 実行中のジョブをキャンセル]**」を選択します。

### 解約行動

1. ジョブは、現在のステップの最後で停止します。
2. その仕事は一歩を踏み出したところで止まらない。
3. 完了までに数分または数時間かかるステップもあります。
4. この間、ジョブは&#x200B;**[!UICONTROL キャンセル]**&#x200B;状態のままになる可能性があります。

> [!NOTE]
>
> 現在のステップが完了すると、ジョブが停止したときに大幅に遅れる可能性があることを理解して、キャンセルを計画します。

## ジョブ履歴の表示

- 過去30日間に実行されたジョブを表示するには：

   1. [!DNL Global Admin Console]にログインします。
   2. 「**[!UICONTROL ジョブ実行]**」を選択します。
   3. ページの一番下までスクロールします。
   4. **[!UICONTROL 最近のジョブ]**&#x200B;を選択します。

- 最近のジョブの表示：
   - **件のジョブ コマンドを送信しました**。
   - 実行に関連する&#x200B;**エラー**&#x200B;および&#x200B;**警告**&#x200B;が発生しました。

> [!NOTE]
>
> 関連オブジェクト **の後の名前の変更や削除は、ジョブ履歴でのコマンドの表示方法**&#x200B;に影響しません。 履歴は、送信時の状態を反映します。
