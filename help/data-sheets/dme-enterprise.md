# Creative CloudとAdobe Document Cloudのエンタープライズサポート

![アイコン](assets/EnterpriseBanner.png)

Adobeは、Adobeのエンタープライズサブスクリプションの一部として含まれる、ビジネスをサポートするための包括的な技術リソースを提供します。 この機能は、エンタープライズサポートプランで強化されます。 エンタープライズのお客様は、特定の技術担当者がAdobeサポートチームにおいて、お客様と技術チームと協力して、すべてのサポートリクエストをタイムリーに解決するためのサービスを提供します。

また、サポートチームは、最も重要な時期にビジネスの中断を最小限に抑えるために、追加のエンタープライズ向けメリットの調整と配信を支援できます。 エンタープライズサポートプランをご利用のお客様は、アドビの詳細で詳細な技術製品ドキュメントと最新のリリースノートを活用することもできます。

## エンタープライズサポートの特長

  <td>

    </div>
  </td>

    <p><i>現地時間に依存します。</i></p>
    </div>

    <p><i>現地時間に依存します。</i></p>
