ジャーニーアクティビティの概要

イベント、オーケストレーション、アクションなどのアクティビティを学習して、複数の手順から成るクロスチャネルのシナリオを作成します。

[ 一般イベントについて説明します ](../using/building-journeys/general-events.md)。

反応イベント

メッセージに対するユーザーのインタラクションを追跡し、反応イベントを設定して、応答をパーソナライズします。

[ 反応イベントについて学ぶ ](../using/building-journeys/reaction-events.md)
:::

:::
![ アイコン ](https://cdn.experienceleague.adobe.com/icons/bullseye.svg?lang=ja)

プロファイルの入口と出口を使用して、ジャーニーを効果的にトリガーまたは進行させます。

[ オーディエンスの選定イベントについて学ぶ ](../using/building-journeys/audience-qualification-events.md)
:::

:::
![ アイコン ](https://cdn.experienceleague.adobe.com/icons/code-branch.svg?lang=ja)

条件アクティビティ

時間やデータ条件などの特定の条件を使用して、ジャーニーでパーソナライズされたパスを定義します。

[ 条件アクティビティについて学ぶ ](../using/building-journeys/condition-activity.md)
:::

:::
![ アイコン ](https://cdn.experienceleague.adobe.com/icons/clock.svg?lang=ja)

待機アクティビティ

正確なタイミングを実現するには、期間またはカスタムの待機オプションを使用して、ジャーニーの遅延を設定します。

[ 待機アクティビティについて説明します ](../using/building-journeys/wait-activity.md)
:::
