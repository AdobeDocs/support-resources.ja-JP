# Creative CloudとAdobe Document Cloudのエリートサポート

![アイコン](assets/Elitebanner.png)

Adobeは、Adobeのエンタープライズサブスクリプションの一部として含まれる、ビジネスをサポートするための包括的な技術リソースを提供します。 これは、ELITE サポートプランで強化されます。 ELITE のお客様は、Adobeサポートチーム内の指定された技術担当者として、お客様と協力してクラス最高のプロアクティブで事後対応的なサポートを提供する、特定サポートエンジニアおよびテクニカルアカウントマネージャにアクセスできます。

適用可能なCreative CloudおよびDocument Cloudソリューションの豊富な経験を活かし、サポートチームは、サポートニーズがどれほど複雑でも、Adobeサポートチームが皆様と協力して、該当するAdobeソリューションへの投資を最大限に活用し、問題が発生する前に回避します。 また、詳細で詳細な技術製品ドキュメントや最新のリリースノートを活用することもできます。

## エリートサポートの特長

    <p>エリートサポート体験の監視と提唱を行い、ビジネス価値を最大化するプロアクティブなサービスを提供する、指定されたテクニカルアカウントマネージャー。</p>
    </div>
  </td>
  <td>

    <p>お客様のソリューション環境およびビジネス目標をよく理解している、専任のサポートエンジニアです。豊富な経験を活かして、お客様のエリートサポートエクスペリエンスの調整を支援します。</p>
